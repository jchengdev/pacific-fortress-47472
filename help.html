Warbler API GUIDE

  • GET / -> API guide
  • GET /about -> Dev notes
  • GET /api/messages -> retrieve all messages (signin JWT required)
  • POST /api/auth/signup -> body: { "username": "username", "email": "some@email.com", "password": "123456", "profileImageUrl": "https://someurl.com" } **JWT in response
  • POST /api/auth/signin -> body: { "email": "some@email.com", "password": "123456" } **JWT in response
  • POST /api/users/<user_id>/messages/ -> create a single message (signin JWT w/ authorization required)
  • GET /api/users/<user_id>/messages/<message_id> -> get a single message (signin JWT w/ authorization required)
  • DELETE /api/users/<user_id>/messages/<message_id> -> delete a single message (signin JWT w/ authorization required)
  • GET /<anything_else> -> 'Not Found' JSON response
